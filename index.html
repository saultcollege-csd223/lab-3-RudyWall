Getting Started with Three.js

This library is used to create 3D shapes and images on a webpage. This library is similar to WebGL. Three.js does use WebGL to make the 3D content but using WebGL alone usually leads to lots of code from the programmer where Three.js handles all scenes, lights, shadows, material, texture and 3D math. Using Three.js makes it better for programmers trying to achieve 3D content with less code.
