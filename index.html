Getting Started with Three.js

Like this fancy cube? Learn to make it!

Why use Three.js:

Three.js is perfect for smooth 3D and 2D animations. It can also be used for interactive graphics. Using Three.js avoids using multi-dimentional objects and if you know another 3D library this library will seem familiar. Three.js uses WebGL to make the 3D content and handles all scenes, lights, shadows, material, texture and 3D math.

Similar Libraries:

WebGL alone requires more code to achieve the same thing Three.js does. Some other libraries that are similar are Unity for higher quality, BabylonJS for quicker functions and AngularJS for static documents.
